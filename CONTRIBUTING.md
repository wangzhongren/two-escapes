# 如何参与贡献

欢迎加入《智能即分类》的开源写作！本项目接受以下形式的贡献：

## 1. 文字改进
- 修正笔误或表述不清之处（提交PR至 `book/`）
- 补充历史/技术细节（需附参考文献）

## 2. 实验扩展
- 在 `experiments/` 添加新验证实验
- 改进现有代码（如增加PyTorch/TensorFlow实现）

## 3. 插图创作
- 使用矢量工具绘制新示意图（存入 `figures/`）
- 改进现有图像清晰度或标注

## 4. 多语言翻译
- 在 `translations/[lang]/` 创建新语言目录
- 保持与主干内容同步

## 提交流程
1. Fork 本仓库
2. 创建特性分支（`git checkout -b feat/new-experiment`）
3. 提交更改（`git commit -am 'Add XYZ'`）
4. 推送至分支（`git push origin feat/new-experiment`）
5. 创建 Pull Request

> 所有贡献默认采用 CC BY 4.0 许可。
